# ParadeOne Architecture

## Purpose
ParadeOne is a parade operations platform for managing registration, lineup, staging, geofenced check-in, section release, live location tracking, announcer scripts, judging, and real-time parade coordination.

## Core Entities
- Organization
- Event
- Entry
- Registration
- Section
- Zone
- Staging Spot
- Route
- User
- Role
- Check-In
- Location Update
- Message
- Announcer Script
- Judge Score

## Core Roles
- Platform Admin
- Organization Admin
- Parade Director
- Section Captain
- Volunteer
- Participant
- Announcer
- Judge

## Event Lifecycle
1. Draft
2. Registration Open
3. Registration Closed
4. Organizer Review
5. Lineup Building
6. Lineup Locked
7. Assignments Sent
8. Check-In Open
9. Parade Active
10. Awards / Results
11. Archived
