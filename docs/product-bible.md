# ParadeOne Product Bible

## Mission
ParadeOne exists to eliminate the chaos of parade operations by giving organizers one unified system to manage registration, staging, communication, live operations, judging, and reporting.

## Core Philosophy

### 1. Think like a parade director
The software should match how organizers actually run a parade.

### 2. Every screen answers a question
Examples:
- Who is registered?
- Who is missing?
- Which section moves next?
- Where is each entry?
- When will they reach Main Street?

### 3. Mobile first
Participants, volunteers, and section captains will mostly use phones.

### 4. Mission Control matters
The organizer dashboard is not just a dashboard. It is ParadeOne Mission Control.

### 5. Real-time awareness beats paperwork
The app should reduce radio traffic, clipboard confusion, and last-minute chaos.

## Core Product Modules

- Create Parade Wizard
- Registration
- Organizer Review
- Lineup Builder
- Staging Spots
- GPS Check-In
- Section Release
- Live Map
- SMS Messaging
- Announcer View
- Judge View
- Reports

## First Build Goal
Build the Create Parade Wizard and Mission Control shell.
